MATLAB Installation
====================
.. warning:: Before installing MATLAB make sure that you are inside your virtual machine running Ubuntu 20.04. 

1. Go to `MATLAB <https://www.mathworks.com>`_
2. Create an account using your official University of Arizona email address (CAT MAIL).
3. Download the installer for MATLAB R2021a. 
4. Select Linux operating system. 
5. Run the installer for MATLAB 
6. Once you installed matlab, add an alias in .bashrc for easy access to MATLAB. 
:: 
    alias matlab=”/usr/local/MATLAB/R2021a/bin/matlab”
